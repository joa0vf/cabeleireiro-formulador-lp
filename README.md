# Cabeleireiro Científico Formulador (CCF)

Este é o repositório oficial da Landing Page do treinamento **Cabeleireiro Científico Formulador**, liderado pela **Profª Neide Dólia**. O projeto visa transformar cabeleireiros em especialistas em cosmetologia capilar e formulação de produtos.

## 🚀 Tecnologias Utilizadas

- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS** (Estilização Moderna e Responsiva)
- **Framer Motion** (Animações Premium e Micro-interações)
- **Lucide React** (Ícones)
- **Vimeo SDK** (Integração de Vídeos de Depoimentos)

## 🛠️ Como Rodar Localmente

### Pré-requisitos
- Node.js (v18 ou superior)
- npm ou yarn

### Passos
1. **Clonar o repositório:**
   ```bash
   git clone [URL-DO-REPOSITORIO]
   cd cabeleireiro-cientifico-formulador
   ```
2. **Instalar as dependências:**
   ```bash
   npm install
   ```
3. **Configurar variáveis de ambiente:**
   - Crie um arquivo `.env.local` na raiz do projeto.
   - Use o `.env.example` como referência.
4. **Executar o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
5. **Acessar o projeto:**
   Abra [http://localhost:3000](http://localhost:3000) no seu navegador.

## 📁 Estrutura do Projeto

- `/app`: Roteamento e layout global.
- `/components`: Componentes reutilizáveis (Hero, Testimonials, Agenda, etc).
- `/public`: Ativos estáticos (Logos em SVG, Imagens e Capas de Vídeo).
- `/hooks` & `/lib`: Lógica e utilitários auxiliares.
